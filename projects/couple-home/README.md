# 🏠 情侣小家 - Couple Home

> 微信小程序 + 网页端 | 情侣同居生活管理工具

## 📱 项目简介

为同居情侣打造的日常生活管理应用，帮助情侣更好地协调生活琐事、记录美好时光。

## 🎯 核心功能

- 🍽️ 吃饭安排（今天吃什么、做饭排班）
- 📅 日程管理（共同日程、提醒）
- 🧹 家务分工（清洁排班、任务追踪）
- 💰 账单 AA（共同支出、分摊计算）
- 📸 照片墙（美好回忆）
- 💝 纪念日（重要日期提醒）
- 📝 购物清单（共同采购）
