### P2 - 轻微 Bug

| 编号 | 描述 | 发现时间 | 状态 |
|------|--------|----------|------|
| BUG-001 | 冰箱添加食材时，日期格式要求严格，需要 `2006-01-02T15:04:05Z07:00` 格式，不支持简单的 `YYYY-MM-DD` | 2026-03-23 16:55 | ⚠️ 已发现 |

**BUG-001 详情:**
- **接口:** `POST /api/fridge`
- **问题:** `expiryDate` 字段使用 `*time.Time` 类型，Go 的 JSON 解析要求完整的 ISO 8601 格式
- **影响:** 用户使用简单日期格式 (`2026-04-01`) 会报错
- **建议修复:** 
  1. 前端使用完整 ISO 格式发送
  2. 或者后端实现自定义时间解析，支持多种格式
