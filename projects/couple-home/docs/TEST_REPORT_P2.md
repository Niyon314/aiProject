### 2. 纪念日 API 测试

| 用例 ID | 测试项 | 请求方法 | 端点 | 预期结果 | 实际结果 | 状态 |
|---------|--------|----------|------|----------|----------|------|
| ANN-001 | 创建纪念日 | POST | `/api/anniversaries` | 返回 201，包含纪念日 ID | ✅ 通过 | ✅ PASS |
| ANN-002 | 获取纪念日列表 | GET | `/api/anniversaries` | 返回纪念日数组 | ✅ 通过 | ✅ PASS |
| ANN-003 | 获取在一起天数 | GET | `/api/anniversaries/days` | 返回天数统计 | ✅ 通过 (374 天) | ✅ PASS |
| ANN-004 | 获取即将到来的纪念日 | GET | `/api/anniversaries/upcoming` | 返回即将到来的纪念日 | ✅ 通过 (空数组) | ✅ PASS |

**测试数据:**
```json
{
  "name": "恋爱纪念日",
  "date": "2025-03-14",
  "icon": "💕",
  "type": "relationship",
  "year": 2025,
  "reminderDays": [7, 3, 1]
}
```

**创建结果:**
- 纪念日 ID: `d14229686b7518a7`
- 在一起天数: **374 天** (从 2025-03-14 到 2026-03-23)

**注:** `anniversaries/upcoming` 返回空数组是因为:
- 初始纪念日 (2025-03-14) 已过去
- 下一周年 (2026-03-14) 也已过去
- 再下一周年 (2027-03-14) 超出默认查询范围
