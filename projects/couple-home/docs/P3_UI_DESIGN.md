## 🎨 全局 UI 规范

### 配色方案

```css
/* AI 菜谱 - 清新绿 */
--ai-chef-primary: #5DBF7D;
--ai-chef-bg: #F0FFF4;

/* 观影清单 - 深夜蓝 */
--movie-primary: #4A90D9;
--movie-bg: #1A1A2E;

/* 恋爱日记 - 浪漫粉 */
--diary-primary: #FF6B81;
--diary-bg: #FFF5F7;

/* 惊喜提醒 - 神秘紫 */
--surprise-primary: #9B59D0;
--surprise-bg: #F8F0FF;
```

### 组件规范

**卡片设计:**
- 圆角：16px
- 阴影：0 2px 8px rgba(0,0,0,0.1)
- 内边距：p-4
- 间距：mb-4

**按钮设计:**
- 主按钮：渐变背景 + 白色文字
- 次按钮：描边 + 主题色文字
- 幽灵按钮：透明背景 + 灰色文字

**图标使用:**
- 尺寸：text-2xl (24px)
- 颜色：与主题色一致
- 库：Lucide Icons

## 📱 响应式规范

### 移动端 (375px - 768px)
- 单列布局
- 卡片占满宽度
- 按钮高度 ≥ 44px

### 平板 (769px - 1024px)
- 双列布局
- 卡片最大宽度 500px
- 居中显示

### 桌面 (1025px+)
- 三列布局
- 卡片最大宽度 400px
- 网格排列
