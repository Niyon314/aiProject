- ❌ 不要所有活都自己干
- ✅ 分配任务给合适的子代理
- ✅ 并行执行多个任务
- ✅ 等待子代理汇报后汇总

**任务分配原则：**
- API 测试 → agent-qa
- 前端开发 → agent-dev
- 后端开发 → agent-dev
- 产品需求 → agent-product
- 部署运维 → agent-ops
- 调研分析 → agent-research

- 汇总各代理结果
