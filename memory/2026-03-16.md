- **平台**: 微信小程序 + 网页端 (Taro 多端)

### 核心功能 (P0 MVP)
1. 用户系统 - 微信登录、情侣绑定
2. 日程管理 - 共享日历、提醒
3. 家务分工 - 清单、轮值、打卡
4. 账单 AA - 记录、自动计算
5. 纪念日 - 管理、倒计时、提醒
6. 照片墙 - 共享相册
7. 吃饭安排 - 今日吃什么
8. 消息通知

### 技术栈
- 前端：Taro + React
- 后端：NestJS + TypeScript + Prisma
- 数据库：PostgreSQL + Redis
- 存储：腾讯云 COS / 阿里云 OSS
- 部署：Docker + GitHub Actions
