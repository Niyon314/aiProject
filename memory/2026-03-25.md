### 踩坑记录
1. **TypeScript 严格模式**: `noUnusedLocals` 连 `_` 前缀变量也不放过，需要直接删除
2. **useState 类型推断**: 初始值字面量类型被锁定，需要显式声明 `useState<{...}>` 类型
3. **Docker buildkit 缓存**: `--no-cache` 对 buildkit 不完全生效，需要 `docker builder prune -af` 清理
4. **docker-compose vs docker run**: docker-compose build 会覆盖手动 build 的镜像，改为直接 docker run
